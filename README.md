# chatbot
This program implements a rule-based chatbot using Python, the NLTK Chat library, and regular expressions to provide simple conversational interactions and AI-related information. The chatbot collects and stores user details such as their name, age, and mood throughout the conversation, allowing it to personalise responses. Pattern matching is used to recognise greetings, user introductions, age statements, mood expressions, and common questions about artificial intelligence. The chatbot class extends NLTK’s built-in Chat functionality by overriding the response method to process user input and generate customised replies. Users can describe how they feel, and the chatbot categorises their mood as positive or negative before responding appropriately. It also adapts explanations of AI concepts based on the user’s age group, providing simpler or more detailed descriptions where appropriate. If a question is not recognised, the chatbot falls back to predefined default responses. The conversation continues until the user enters “bye”, after which a personalised farewell message is displayed.
